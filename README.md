
#  Transit Express – Courier & Seat Booking App

A responsive web application for booking transport seats and courier services across Kenya. Built with HTML, Tailwind CSS, and vanilla JavaScript, this app supports dynamic pricing, seat locking, and ticket persistence.

---

##  Project Pages

- `index.html`
- `login.html`
- `booking.html`
- `courier.html`
- `payseat.html`
- `paycourier.html`
- `sign.html`
- `routes.html`
- `viewtickets.html`
---

##  Login Page

- Email and password fields
- Buttons: **Login**, **Sign Up**, **Forgot Password**
- Navigation: Home, Routes, Courier Services, Sign In
- Footer: Transit Express 2026

---

## Courier Booking

- Weight-based pricing
- Surcharge for parcels over 30kg
- Dynamic price breakdown shown before payment

---

##  Seat Booking

- Route, day, time, seat, and price stored in `localStorage`
- Confirmation page (`payseat.html`) displays ticket details dynamically
- Booking data is parsed and injected using JavaScript

---

##  Business Logic Highlights

- Seat locking prevents double-booking
- Ticket confirmation persists across sessions
- Refund and cancellation logic planned
- API-ready structure for future backend integration

---

##  Tech Stack

- **Frontend**: HTML, Tailwind CSS
- **Logic**: Vanilla JavaScript
- **Design**: Modular layout, responsive cards, dropdown filters
- **Tools**: VS Code, MSYS2 terminal, Chrome for live testing
